# adas_aeb
Repository containing all of the aeb functionality files

test_lib.cpp requires CppLinuxSerial library to run properly (link: https://github.com/gbmhunter/CppLinuxSerial)

Basically both files achieve same result through different means. For them to work properly it is necessary to grant user appropriate permissions. The easiest way to achieve that is through adding said user to the dialout group.
